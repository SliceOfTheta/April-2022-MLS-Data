Creating New Variables: OLP_Median & CP_Median¶

I want to explore the relationship sale price has depending on if it is higher than the median or lower than the median. I will create two new columns in my data set for two seperate categories: OLP_Median & CP_Median.

In [10]:
median_Original = df["Original_List_Price"].median() 

# create a list of our conditions
conditions = [ 
    (df["Original_List_Price"] <= median_Original),
    (df["Original_List_Price"] > median_Original)
]

# create a list of the values we want to assign for each condition
values = ['Lower_Median', 'Higher_Median']

# create a new column and use np.select to assign values to it using our lists as arguments
df['Median_OP'] = np.select(conditions, values)
